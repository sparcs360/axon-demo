# About

Yet another example of an enterprise application architecture built upon the [Axon 3 Framework](http://www.axonframework.org/)... with a little help from:

- Back-end
  - Spring Framework
  - Spring Boot
  - Spring Data (JPA)
  - Spring Messaging
  - Docker

- Databases
  - H2
  - MySql

- Messaging
  - RabbitMQ

- Front-end
  - Bootstrap
  - Angular.JS (v1)
  - ngStomp
